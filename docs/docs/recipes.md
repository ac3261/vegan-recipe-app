## Seasonings to Feature

Optional: highlight herbs, spices, or condiments you want the chef to work in.

### Adding Seasonings

To add a seasoning, type its name in the search bar and select it from the dropdown list. You can also type a custom seasoning name and press Enter to add it.

### Removing Seasonings

To remove a seasoning, click the 'X' button next to it in the list.

### Seasoning Limit

You can select up to 12 seasonings for the chef to work with.
